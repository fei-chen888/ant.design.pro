### 2.1 关于baseURL的配置

    baseURL文件指明了后端服务地址 本项目采用前后端分离设计，前端以据此URL访问后台服务
|字段名|值|备注|
|----|---|---|
|baseURL|https://msa-test-server.meizhidev.com|前端以据此URL访问后台服务|

``` shell
文件位置
msa-tenant
    |-- src
        |-- api
            |-- baseURL.ts
            ...
        ...
    ...
```
``` js
// 示例：
export const baseURL = 'https://msa-test-server.meizhidev.com'
```
